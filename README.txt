Привет, перед тобой локальная версия пакмана с ограниченным набором возможностей

Чтобы запустить игру потребуется Node.js LTS версии (16.13.0) или свежее,
скачать можно здесь: https://nodejs.org/en/download/

Для запуска можешь использовать команду npm start или node main.js
Код своего решения можешь писать в файле task.js

- Чтобы изменить уровень передай в функцию run другие настройки из файла settings.js
- Чтобы изменить количество разрешенных ходов, используй параметр turnLimit
- Чтобы изменить время ожидания между ходами, используй параметр turnTimeMs,
  передай 0 чтобы полностью отключить ожидание
- Чтобы управлять показом финальных результатов и последних нескольких шагов,
  используй параметр showTestOutput
